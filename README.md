# Generador de Claves en Escritura Leet

Este es un programa de consola en .NET Core que genera claves en escritura "leet" (reemplazando caracteres por números y símbolos). Simplemente ingresa un texto o frase, y el programa generará una clave en escritura "leet" junto con caracteres aleatorios para alcanzar la longitud deseada.

## Instrucciones de Uso

1. Clona este repositorio o descarga los archivos.
2. Abre el proyecto en tu entorno de desarrollo preferido (por ejemplo, Visual Studio, Visual Studio Code).
3. Ejecuta la aplicación y sigue las instrucciones en la consola para generar una clave en escritura "leet".

## Pasos
1. Al ejecutar el programa, verás un mensaje de bienvenida y una solicitud para ingresar un texto o frase.

2. Ingresa el texto o frase deseado y presiona Enter.

3. La aplicación generará una clave en escritura "leet" basada en tu entrada, junto con caracteres aleatorios para alcanzar la longitud deseada.

4. La clave generada se mostrará en la consola.

5. Presiona cualquier tecla para salir de la aplicación.

## Personalización
Puedes personalizar el mapeo "leet" y la generación de caracteres aleatorios ajustando el código en la función **"GenerateClave"**. 
Modifica el diccionario **"leetMap"** para cambiar cómo se reemplazan los caracteres y ajusta las opciones de generación aleatoria según tus preferencias.
